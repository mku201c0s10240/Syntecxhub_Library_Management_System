# Library Management System - Java (CLI)
A simple Command-Line Interface (CLI) based Library Management System built using Core Java, File Handling, and Exception Handling.This project allows users to add, remove, search, and display books
using a text file (books.txt) as storage.

# Features
1. Add new books (ID, Title, Author)
2. Remove books by ID
3. Search books by ID
4. Display all books
5. Uses file handling (BufferedReader, BufferedWriter)
6. Includes exception handling for safe input validation
7. Lightweight, fast, and beginner-friendly
